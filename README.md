# Bumi Borneo T-Shirt Printing Website

This is the official website for **Bumi Borneo T-Shirt Printing**, built with [Vue.js](https://vuejs.org/) and deployed on [Vercel](https://vercel.com/).

The project serves as a landing page and product showcase for the business, providing details about services, promotions, and contact information.
